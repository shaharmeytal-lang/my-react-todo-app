# Todo App — React 19 + Vite

## תיאור קצר של האפליקציה
אפליקציית מנהל משימות (Todo App) בנויה ב-React 19 ו-Vite. האפליקציה מאפשרת למשתמש להוסיף, לערוך, למחוק ולסנן משימות. המשימות נשמרות ב-localStorage כך שהן נשמרות גם לאחר סגירת הדפדפן.

**טכנולוגיות:**
- React 19 (קומפוננטות פונקציונליות בלבד)
- Vite (build tool)
- JavaScript ES6+
- localStorage (persistence)
- CSS בסיסי

## הוראות להרצת הפרויקט מקומית

1. **התקנת תלויות:**
```bash
npm install
```

2. **הרצת development server:**
```bash
npm run dev
```
פתח בדפדפן: `http://localhost:5173/` (או הפורט שיודפס בטרמינל)

3. **בנייה ל-production (אופציונלי):**
```bash
npm run build
```

## קומפוננטות ואחריותן

- **App.jsx** — קומפוננטת שורש הנושאת את ה-state הראשי של המשימות, מטפלת בשמירה/שחזור מ-localStorage, ומעבירה callbacks לילדים.
- **AddTodo.jsx** — טופס מבוקר להוספת משימות חדשות.
- **TodoList.jsx** — מציגה רשימת משימות מסוננות ומעבירה callbacks לכל פריט.
- **TodoItem.jsx** — מציגה משימה בודדת עם אפשרויות toggle, edit inline ו-delete.
- **FilterBar.jsx** — מציגה כפתורי פילטר (All / Active / Completed) וספירת משימות פעילות.

## מגבלות וביאגים ידועים

- **מזהים משימות:** משתמש ב-Date.now() + random string קצר; מספיק לפרויקט לימודי אך לא 100% קל לחזק בייצור.
- **localStorage בלבד:** אין סינכרון בין טבים/חלונות שונים; כל כרטיסיה לוקחת את ה-state שלה בעצם.
- **ללא validation טוב יותר:** בדיקה בסיסית של טקסט ריק; אין validation של אורך או תווים מיוחדים.
- **ללא עריכה bulk:** לא ניתן לבחור מספר משימות ולמחוק/לסמן אותן בו-זמנית.
- **ללא drag-and-drop:** סדר המשימות קבוע לפי סדר יצירה (פשוט להוסיף בעתיד).
- **ללא search/filter טוקסט:** אפשר לסנן לפי completed או active, אך לא לחפש לפי טקסט.
